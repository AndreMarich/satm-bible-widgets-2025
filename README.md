# SATM Bible Widgets

Interactive Bible Scripture Selector and Audio Bible Player for the SATM Driver App.

## Files

- `scripture-selector.html` - Scripture lookup and display widget
- - `audio-bible.html` - Audio Bible chapter player
 
  - ## Hosted URLs
 
  - These files are hosted via GitHub Pages at:
 
  - - Scripture Selector: `https://AndreMarch.github.io/satm-bible-widgets-2025/scripture-selector.html`
    - - Audio Bible: `https://AndreMarch.github.io/satm-bible-widgets-2025/audio-bible.html`
     
      - ## Features
     
      - ### Scripture Selector
      - - Browse all 66 books of the Bible
        - - Select any chapter
          - - Choose specific verses
            - - Display scripture text from the BSB (Bible Standard Version)
              - - Beautiful responsive design
               
                - ### Audio Bible Player
                - - Listen to any Bible chapter
                  - - Select from all 66 books
                    - - Built-in HTML5 audio player
                      - - Responsive and user-friendly interface
                       
                        - ## Setup Instructions
                       
                        - 1. Create a new GitHub repository named satm-bible-widgets
                          2. 2. Upload these two HTML files to the repository
                             3. 3. Go to repository Settings > Pages
                                4. 4. Set Source to main branch and / (root)
                                   5. 5. Your files will be live at the URLs above
                                     
                                      6. ## API Sources
                                     
                                      7. - Scripture text: Free Use Bible API (BSB version)
                                         - - Audio: Bible.com public audio resources
